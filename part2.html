۱۱ خروجی کد زیر چیست؟ 

text = 'hello'
print(text.find('o', 0, 3))
4

3

توضیح: متد .find() در یک محدوده مشخص شده جستجو می‌کند. در اینجا، به دنبال کاراکتر 'o' در برش text[0:3] که معادل 'hel' است، می‌گردد. چون 'o' در این محدوده پیدا نمی‌شود، متد مقدار -1 را برمی‌گرداند.

۱۲ طبق قواعد زبان پایتون، کدام یک از تعریف متغیرهای زیر غیر مجاز است؟

var_1 = 10
_var_2_ = 20
_ = 30
4_var = 40
توضیح: در پایتون، نام متغیرها نمی‌تواند با عدد شروع شود. سایر گزینه‌ها (شروع با حرف یا آندرلاین) کاملا مجاز هستند. بنابراین گزینه 4_var باعث خطای SyntaxError می‌شود.

۱۳ مقدار نهایی count پس از اجرای کد زیر چند خواهد بود؟ 

count = 0
for i in range(3):
  for j in range(2):
    count += i + j
9
6
5
12
توضیح: مقادیر `i` برابر 0, 1, 2 و مقادیر `j` برابر 0, 1 خواهند بود. مجموع `i+j` در هر تکرار به `count` اضافه می‌شود:
(0+0) + (0+1) + (1+0) + (1+1) + (2+0) + (2+1) = 0 + 1 + 1 + 2 + 2 + 3 = 9

۱۴ خروجی قطعه کد زیر کدام است؟ 

print(bool(0), bool(3.14), bool(-3), bool((18 - 3 + 3 * 5)/2))
False False True True
False False True False
False True True True
True False True False
توضیح:
- `bool(0)` برابر `False` است چون صفر مقدار Falsy محسوب می‌شود.
- `bool(3.14)` و `bool(-3)` هر دو `True` هستند چون اعداد غیر صفر Truthy هستند.
- عبارت آخر `(18 - 3 + 15)/2` برابر `30/2` یا `15` می‌شود که عددی غیر صفر و `True` است.

۱۵ خروجی قطعه کد زیر کدام است؟ 

def first_func(num):
    i = 3
    while True:
        i *= 2
        if i > num:
            return i
def second_func(num):
    count = 0
    while True:
        num /= 2
        count += 1
        if num < 5:
            return count
print(first_func(30) * second_func(20))
Syntax Error
144
96
60
توضیح:
- `first_func(30)`: متغیر `i` با مقدار اولیه 3، به ترتیب 6، 12، 24 و 48 می‌شود. چون 48 بزرگتر از 30 است، تابع مقدار `48` را برمی‌گرداند.
- `second_func(20)`: متغیر `num` به ترتیب 10، 5 و 2.5 می‌شود. در هر مرحله `count` یکی اضافه می‌شود. وقتی `num` به 2.5 می‌رسد (کوچکتر از 5)، تابع مقدار `count` که `3` است را برمی‌گرداند.
- در نهایت `48 * 3` محاسبه و `144` چاپ می‌شود.

۱۶ خروجی کد زیر کدام خواهد بود؟ 

a, b, count = 1, 1, 1
n = 55
while b < n:
    a, b = b, a + b
    count += 1
print(count)
9
10
55
TimeLimit
توضیح: این کد دنباله فیبوناچی را تولید می‌کند و شمارنده را افزایش می‌دهد. حلقه تا زمانی ادامه دارد که `b` (عدد فعلی فیبوناچی) کوچکتر از 55 باشد. دنباله `b`: 1, 2, 3, 5, 8, 13, 21, 34, 55. وقتی `b` به 55 می‌رسد، شرط `55 < 55` برقرار نیست و حلقه متوقف می‌شود. در این لحظه `count` به `9` رسیده است.

۱۷ خروجی کد زیر کدام گزینه خواهد بود؟ 

n = 64
count = 1
for i in range(2, n):
    root = i**2
    if root == n:
        count += 1
    else:
        while root < n:
            root *= i
            if root == n:
                count += 1
print(count)
2
4
3
8
توضیح: این کد تعداد توان‌های کاملی که برابر 64 هستند را پیدا می‌کند (به علاوه‌ی 64 به توان 1).
- `count` با 1 شروع می‌شود (برای 64¹).
- برای `i=2`: 2⁶ برابر 64 است. `count` می‌شود 2.
- برای `i=4`: 4³ برابر 64 است. `count` می‌شود 3.
- برای `i=8`: 8² برابر 64 است. `count` می‌شود 4.
در نهایت مقدار 4 چاپ می‌شود.

۱۸ خروجی قطعه کد زیر کدام است؟ 

x = 25
y = 37.7
print('x = %.1f | y = %d' % (x, y))
x = 25.0 | y = 38
x = 25 | y = 37.7
x = 25 | y = 38
x = 25.0 | y = 37
توضیح: این کد از روش قالب‌بندی قدیمی پایتون استفاده می‌کند.
- `%.1f` مقدار اول (`x`) را به عنوان یک عدد اعشاری با یک رقم دقت نمایش می‌دهد (25 تبدیل به 25.0 می‌شود).
- `%d` مقدار دوم (`y`) را به عنوان یک عدد صحیح نمایش می‌دهد و بخش اعشاری آن را حذف می‌کند (37.7 تبدیل به 37 می‌شود).

۱۹ خروجی قطعه کد زیر کدام است؟ 

s1 = {1, 2, 3}
s2 = {3, 4, 5}
print(s1.union(s2) == s2.union(s1))
True
False
SyntaxError
{1, 2, 3, 4, 5}
توضیح: عمل اجتماع (union) در مجموعه‌ها دارای خاصیت جابجایی است. این یعنی ترتیب اعمال آن در نتیجه تاثیری ندارد. `s1.union(s2)` و `s2.union(s1)` هر دو مجموعه یکسان `{1, 2, 3, 4, 5}` را تولید می‌کنند. مقایسه دو مجموعه یکسان با `==` نتیجه `True` را برمی‌گرداند.
